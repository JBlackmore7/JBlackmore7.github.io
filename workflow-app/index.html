Work Order Flow Application

This will be an application used to manage the creation, editing, flow, and archiving of work orders throughout the company.

Why is This Application Necessary?

To update our archaic paper work order system and to reduce the amount of human error associated with the transfer of physical work orders.

Features of This Application

  • Create a work order
  • Edit a work order
  • Assign a work order to a user
  • Searchable work order archive
